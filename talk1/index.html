~ talk title

Linux Internal: eBPF

~ talk sypnosis

Extended Berkeley Packet Filter (eBPF) is a revolutionary technology with origins in the Linux kernel that can run sandboxed programs in a privileged context such as the operating system kernel.

In this presentation, we will briefly explain how eBPF works and the advantages of eBPF.
